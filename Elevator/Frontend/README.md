# Elevator Control System — Frontend

React 18 + Vite + Tailwind dashboard for the Elevator Simulation backend.

## Run
```
npm install
npm run dev
```

App runs at http://localhost:5173 and expects backend at http://localhost:4000.

## Routes
- `/` Dashboard
- `/elevators`, `/elevators/:id`
- `/requests`
- `/strategy`, `/logs`, `/settings`
