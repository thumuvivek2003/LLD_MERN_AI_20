USER APIs

User	Home / Browse Movies	GET /api/movies	Get movies list
User	Home / Search	GET /api/movies?search=leo	Search movies
User	Home / Filter by City	GET /api/movies?city=hyderabad	Filter movies
User	Movie Details	GET /api/movies/:movieId	Get movie details
User	Movie Details	GET /api/movies/:movieId/shows	Get movie shows
User	Theater & Timings	GET /api/shows/:showId	Get show details

User	Seat Selection	POST /api/seat-locks	Lock selected seats
User	Payment	POST /api/payments/process	Dummy payment
User	Booking Success	POST /api/bookings	Confirm booking
User	My Bookings	GET /api/bookings/me	User booking history
User	Booking Details	GET /api/bookings/:bookingId	Single booking
