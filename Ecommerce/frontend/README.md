# ShopKart Frontend

React + Vite + Tailwind frontend for the Ecommerce Cart & Checkout MVP.

## Run

1. `npm install`
2. Ensure backend is running on `http://localhost:5001`
3. `npm run dev` (serves on `http://localhost:5173`)

## Seeded credentials

- Admin: `admin@shop.com` / `admin123`
- Customer: `user@shop.com` / `user123`

## Env

Set `VITE_API_URL` in `.env` (default `http://localhost:5001/api`).

## Scripts

- `npm run dev` — local dev server
- `npm run build` — production build
- `npm run preview` — preview built bundle
