# Ecommerce Backend (MVP)

LLD-focused MERN backend for cart, coupon, pricing, checkout, payment, order.

## Run

1. `npm install`
2. Ensure local MongoDB is running (`mongodb://127.0.0.1:27017`).
3. `npm run seed`
4. `npm run dev`

API mounts under `http://localhost:5001/api`.

## Seeded credentials

- Admin: `admin@shop.com` / `admin123`
- Customer: `user@shop.com` / `user123`
